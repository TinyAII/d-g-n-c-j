# AstrBot Essential

为 AstrBot 支持基本的娱乐功能。

支持功能：
- 腾讯元宝助手
- DeepSeek-3.2助手
- DeepSeek-3.1助手
- GPT5-nano助手
- Claude4.5-hiku助手
- 通义千问助手
- DeepSeek-R1助手
- 智谱GLM4.6助手
- 夸克AI助手
- 蚂蚁AI助手
- 豆包AI助手
- ChatGPT-oss助手
- 谷歌Gemini-2.5助手
- 阿里AI助手
- 讯飞AI助手
- 小米MiMo-V2助手

安装方法：
```
plugin i d-g-n-c-j
```

使用方法：
- `腾讯元宝 <提问内容>`：调用腾讯元宝助手进行问答
  - 支持数学计算、知识问答、闲聊等

  - 示例：`腾讯元宝 你是谁`
